This project aims to develop an AI chat robot using Chatterbot technology, speech recognition libraries in Python, and Google's APIs to assist students in the College of Engineering and Information Technology. The robot has a sensor system that detects when a student has a question, and they can ask their question through the robot's microphone. The spoken question is then converted into text using Google's Speech-to-Text API. The AI module, built using Chatterbot, analyses the question and retrieves the answer from its database. The response is converted back to speech using Google's Text-to-Speech (gTTS) API and output through the robot's speaker. The robot provides specialized information on computer systems engineering to help students find answers to their queries. The project involves building the AI module, implementing speech recognition, integrating Google's API, designing and building the sensor system, and developing the robot. Testing and refinement of the system are necessary to ensure its functionality and performance. The project will benefit the college's students and visitors by providing them with an accessible and reliable source of information.
